Chess Game in C++

A simple console-based Chess Game developed in C++ using Object-Oriented Programming concepts. The project includes chess board setup, piece movement, turn handling, and move validation.

Features
8×8 Chess Board
Two-player gameplay
Valid move checking
Different classes for chess pieces
Turn-based system
OOP concepts implementation
OOP Concepts Used
Encapsulation
Inheritance
Polymorphism
Composition
Technologies
C++
Object-Oriented Programming
Console Application
Project Purpose

This project was created as a semester project for the OOP Lab to practically implement Object-Oriented Programming concepts in C++.
